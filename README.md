# Raytonx Nest Packages

这是 Raytonx 的 NestJS 可复用基础设施模块仓库（monorepo）。每个模块都可以独立安装、独立构建与发布。

English version: `README.en.md`

## 包列表

- `@raytonx/core`：通用类型与工具方法（Nest 模块公共能力）。`packages/core/README.md`
- `@raytonx/config`：配置模块（env 文件加载 + Zod 校验/转换）。`packages/config/README.md`
